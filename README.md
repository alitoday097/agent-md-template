# agent.md Template

A high-signal, minimal `agent.md` template for AI-driven development. Helps AI agents maintain context, follow strict scope, and produce consistent outputs.

---

## What is this?

`agent.md` is a control file for AI coding agents. It defines:

- Project purpose and type  
- Core features and non-goals  
- Data model and UI/UX philosophy  
- Coding standards and execution strategy  
- Agent behavior rules  

By following this template, you reduce **context decay**, avoid unnecessary repetition, and ship clean MVPs faster.

---

## How to Use

1. Copy `agent.md` into your project  
2. Fill in **purpose** and **feature list**  
3. Ensure **scope boundaries** are defined  
4. Let your AI agent read and follow it  

> Works with Claude, Cursor, Windsurf, and similar AI-driven coding tools.

---

## Why It Matters

- Keeps your AI “on track”  
- Prevents feature creep  
- Saves time and reduces errors  
- Standardizes AI behavior across projects
